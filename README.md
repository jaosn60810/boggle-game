# Run Locally

Clone the project

```bash
  git clone https://github.com/jaosn60810/boggle-game.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
